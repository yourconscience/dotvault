# dotvault Private Vault Template

This directory is a generic starter template for a user's private vault. Copy or apply it to a private location such as `~/Workspace/vault`; do not store real private notes in the public `~/Workspace/dotvault` checkout.

## Starter directories

- `notes/` for human-authored notes.
- `memory/` for agent-written memory and digests.
- `profile/` for durable user-selected context.
- `sessions/` for session summaries and handoffs.

The `.gitkeep` files are placeholders so the empty directory layout can be versioned.
